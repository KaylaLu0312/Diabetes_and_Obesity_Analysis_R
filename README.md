# Diabetes_and_Obesity_Analysis_R
A data analysis project examining the prevalence of diabetes and obesity, and the association between these two conditions, using R.

## Tools
- R
- R Markdown
- ggplot2
- dplyr

## Key Analysis
- Diabetes prevalence
- Diabetes prevalence in the Top 20 countries
- Obesity prevalence
- Obesity prevalence in the Top 20 countries

## Data Source
- Dataset_1:  Diabetes Prevalence Dataset
- Source: [Word bank Group_Data 360](https://data360.worldbank.org/en/indicator/WB_WDI_SH_STA_DIAB_ZS)
- Description: Diabetes prevalence refers to the percentage of people ages 20-79 who have type 1 or type 2 diabetes. It is calculated by adjusting to a standard population age-structure.
- Dataset_2:  Obesity Prevalence Dataset
- Source: [WHO](https://www.who.int/data/gho/data/indicators/indicator-details/GHO/prevalence-of-obesity-among-adults-bmi-=-30-(age-standardized-estimate)-(-))
- Description: Prevalence of obesity among adults (aged 18+), defined as BMI ≥ 30.
